To run gas phase code with a sample parameter set:
1. Download chamber_gas_phase.m, Chamber_gas_parameters_filter_PAPER.m (from Additional Codes folder), and Flow_data.mat (from Model Calibration folder)
2. Run Chamber_gas_parameters_filter_PAPER.m
3. Run chamber_gas_phase.m (sample IH protocol provided for IH run)

To run gas-liquid phase code with a sample parameter set:
1. Download chamber_gas_and_liquid_phase.m, Chamber_liquid_parameters_filter_PAPER.m (from Additional Codes folder), and Flow_data.mat (from Model Calibration folder)
2. Run Chamber_liquid_parameters_filter_PAPER.m
3. Run chamber_gas_and_liquid_phase.m (sample IH protocol provided for IH run)
